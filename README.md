# Quiniela Mundial

Aplicación web para una quiniela del Mundial de la FIFA entre ~15 amigos. Predicción de marcadores, preguntas bonus, predicciones de campeón/subcampeón/goleador, tabla de posiciones en tiempo real y panel de administración.

## Stack

- Next.js + TypeScript + Tailwind CSS
- Supabase (Postgres + Auth + Realtime)
- Vercel (hosting + cron)
- football-data.org (API de partidos)

## Desarrollo local

```bash
npm install
npm run dev
```

La app corre en `http://localhost:3000`.
